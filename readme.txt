== Installation ==

1. Upload the `social-sharing-block-gutenberg` folder to your `/wp-content/plugins/` directory or alternatively upload the social-sharing-block-gutenberg.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Install and activate the Gutenberg WordPress plugin.
2. Activate the plugins through the 'Plugins' menu in WordPress.
3. Add your social sharing block within Gutenberg with the new "Social" block.
